# Web_Development_Curriculum
This repository was created by the owner as a means to systematically learn programming and to document their learning journey(please note that the owner is also a beginner).   
The repository is public in the hope that it might provide some assistance to other beginners who may lack resources.   
Through this curriculum, you can explore various programming languages, frameworks, and tools, gaining essential skills and concepts necessary for modern software development.   
It covers a comprehensive range of topics from C programming to computer science and web development.

By following this repository, you will engage in a problem-solving approach using a problem bank, focusing more on practical exercises rather than theoretical study.

Audience:   
This curriculum is intended for beginner to intermediate learners who have a basic understanding of programming and wish to gain practical experience.   
It aims to cover a wide range of topics from foundational programming to intermediate and advanced subjects, allowing learners to progress step-by-step and enhance their skills.

## About owner
The owner of this repo is a high school graduate who dropped out of college and completed a government-funded training program.   
After a six-month educational course, he immediately entered the job market, only to face inevitable failure.   
Although he eventually found employment, he was disheartened by his paper-thin skills.   
Determined to improve, he decided to resign and, after a brief period of rest, began studying programming again.

Realizing the inadequacy of his foundational knowledge, he resolved to start from the basics with the C programming language.   
This repository was created to document his journey and to help other job seekers avoid similar pitfalls.

Will he remain a couch potato, or become a productive member of the workforce?   

What will be the fate of the owner?

## Curriculum Overview
This curriculum is composed of several key stages that cover various aspects of software development.   
Each stage adopts a problem-solving-focused learning approach using a problem bank, enabling the application of theoretical knowledge to real-world scenarios.

1. C Programming
* Master the basics of C programming, including syntax, data types, and control structures.
* Solve problems using fundamental data structures and algorithms.

2. Data Structures
* Deepen understanding of data structures such as arrays, linked lists, stacks, queues, trees, and graphs.

3. Algorithms
* Implement and analyze algorithms for sorting, searching, and optimization.

4. C++ Programming
* Transition to C++ and explore object-oriented programming concepts.
* Cover features like templates, STL, and memory management.

5. Operating Systems, Networking, and Databases
* Study CS subjects such as operating systems, networking, and databases.
* Acquire skills in system programming and database design.

6. Java Programming
* Develop applications in Java with a focus on object-oriented design and best practices.
* Explore Java's rich ecosystem and libraries.

7. Web Development: HTML, CSS, JavaScript
* Build responsive web pages using HTML and CSS.
* Enhance interactivity with JavaScript and modern front-end techniques.

8. Java Web Technologies: Servlet, JSP
* Development dynamic web applications using Java Servlets and JSP.
* Understand the MVC architecture and web application deployment.

9. Spring Framework
* Learn to develop applications with the Spring Framework

10. Spring boot
* Develop applications using Spring Boot, leveraging its auto-configuration and other features.

11. Node.js
* Build server-side applications with Node.js

12. React
* Create interactive UIs with React and learn to manage state effectively.

13. TypeScript
* Enhance JavaScript development with TypeScript's static typing

14. Next.js
* Build server-rendered React applications with Next.js to improve performance and SEO.

15. Cloud Computing with AWS
* Learn to deploy and manage applications on AWS cloud infrastructure.
* Explore services like EC2, S3, Lambda, and RDS.

16. CI/CD
* Implement continuous integration and deployment pipelines.

17. Docker
* Learn to containerize applications with Docker to ensure consistent deployment across environments.

## Learning Approach
This curriculum aims to focus on hands-on learning rather than theory, adopting a problem-solving-centered approach using a problem bank.   
Through this method, learners will progress in the following ways:
* Problem-Solving Focus:   
Learn each topic through a variety of problem-solving-based exercises.   
This approach emphasizes developing the ability to apply theoretical knowledge to solve real-world problems.

* Practice-Oriented Learning:  
Each stage is designed to reinforce learning through practice.   
Through hands-on exercises, learners can apply theory in practice and acquire practical skills.

* Step-by-Step Learning:
The curriculum covers topics from beginner to advanced levels sequentially, allowing learners to start at their level and gradually improve their skills.

* Self-Directed Learning:   
Learning for each topic is conducted in a self-directed manner.   
Learners complete each section at their own pace and use additional resources for deeper learning as needed.

## Progress
Each stage of the curriculum is continuously updated as the owner completes his learning.   
As long as the the owner doesn't quit midway, new content will be consistently added.

## Goals
This curriculum aims to provide practical assistance to learners who wish to systematically study programming.   Through a problem-solving-focused learning approach, it helps learners acquire not only theoretical knowledge but also practical problem-solving skills.   
On a personal note, the goal is to study consistently to either secure a job at a desired company or be able to smoothly create desired web applications.   
Additionally, the aim is to make a positive contribution to the programming ecosystem by helping others avoid unnecessary trial and error.

## 커리큘럼 개요
이 커리큘럼은 소프트웨어 개발의 다양한 측면을 다루는 여러 주요 단계로 구성되어 있다.   
각 단계는 문제은행을 활용한 문제 해결 중심의 학습 접근 방식을 채택하여, 이론적 지식을 실제 시나리오에 적용할 수 있도록 할 것이다.

1. C 프로그래밍
* C 프로그래밍의 기초를 익히고, 문법, 데이터 타입, 제어 구조를 학습.
* 기본적인 자료구조와 알고리즘을 사용하여 문제 해결.

2. 자료구조
* 배열, 연결 리스트, 스택, 큐, 트리, 그래프와 같은 자료구조에 대한 이해를 심화.

3. 알고리즘
* 정렬, 검색, 최적화 알고리즘을 구현하고 분석.

4. C++ 프로그래밍
* C++로 전환하여 객체 지향 프로그래밍 개념 탐구.
* 템플릿, STL, 메모리 관리와 같은 기능 다룸.

5. 운영체제, 네트워크, 데이터베이스
* CS 과목인 운영체제, 네트워크, 데이터베이스에 대해 학습.
* 시스템 프로그래밍과 데이터베이스 설계에 대한 기술 습득.

6. 자바 프로그래밍
* 객체 지향 설계와 모범 사례에 중점을 두고 자바로 애플리케이션을 개발.
* 자바의 풍부한 생태계와 라이브러리를 탐구.

7. 웹 개발: HTML, CSS, JavaScript
* HTML과 CSS를 사용하여 반응형 웹 페이지 구축.
* JavaScript와 최신 프론트엔드 기술로 상호작용 강화.

8. 자바 웹 기술: Servlet, JSP
* Java Servlets 와 JSP를 사용하여 동적 웹 애플리케이션 개발.
*MVC 아키텍처와 웹 애플리케이션 배포를 이해.

9. 스프링 프레임워크
* 스프링 프레임워크로 애플리케이션 개발 학습.

10. 스프링 부트
* 스프링 부트의 자동 구성 등 스프링 부트를 활용한 애플리케이션 개발 학습.

11. Node.js
* Node.js로 서버 측 애플리케이션 구축.

12. React
* React로 상호작용하는 UI를 만들고 상태를 효과적으로 관리하는 법을 학습.

13. TypeScript
* TypeScript의 정적 타이핑으로 JavaScript 개발 향상시키는 방법에 대해 학습.

14. Next.js
* Next.js로 서버 렌더링된 React 애플리케이션을 구축하여 성능과 SEO 개선하는 법에 대해 학습.

15. AWS를 활용한 클라우드 컴퓨팅
* AWS 클라우드 인프라에 애플리케이션을 배포하고 관리하는 법을 배움.
* EC2, S3, Lambda, RDS와 같은 서비스 탐구.

16. CI/CD
* 지속적 통합 및 배포 파이프라인 구현.

17. Docker
* Docker로 애플리케이션을 컨테이너화하여 환경 간 일관된 배포를 보장하는 법 학습.

## 학습 방식
이 커리큘럼은 이론보다는 실습을 통해 학습하는 것을 목표로 하며, 문제은행을 활용한 문제 해결 중심의 접근 방식을 채택했다.   
이를 통해 학습자는 다음과 같은 방식으로 학습을 진행하게 된다.

* 문제 해결 중심:   
문제해결에 기반한 다양한 문제를 통해 각 주제를 학습한다.   
이 방식은 이론적 지식을 실제 문제에 적용하여 해결하는 능력을 키우는 데 중점을 둔다.

* 실습 위주 학습:   
각 단계는 실습을 통해 학습 내용을 강화할 수 있도록 설계되어 있다.   
실습을 통해 학습자는 이론을 실제로 적용하고, 실무적인 기술을 습득할 수 있다.

* 단계별 학습:   
커리큘럼은 초급부터 고급까지의 주제를 순차적으로 다루고 있어, 학습자는 자신의 수준에 맞는 단계부터 시작하여 점진적으로 실력을 향상시킬 수 있다.

* 자기 주도 학습:   
각 주제에 대한 학습은 자기 주도적으로 진행된다.   
자신의 학습 속도에 맞춰 각 섹션을 완료하고, 필요에 따라 추가 자료를 활용해서 심화 학습한다.
